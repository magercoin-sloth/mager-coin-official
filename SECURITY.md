# Security Policy

## Supported Versions
We are committed to ensuring the security of the $MAGER ecosystem. The versions currently supported for security updates are:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅ |

## Reporting a Vulnerability
If you discover a security vulnerability, please do **not** report it publicly through 'Issues'. 

Instead, please contact the development team directly via the official email of PT Lunar Sloth Internasional or send a private message to the Developer.

The security of our community's assets is our top priority.
